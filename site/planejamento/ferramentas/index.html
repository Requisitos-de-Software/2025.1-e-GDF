    Ferramentas

    Introdução

    A fim de facilitar o gerenciamento e a execução do projeto, foram utilizadas certas ferramentas com o intuito de facilitar certas atividades e produção de artefatos. As ferramentas selecionadas para a realização do projeto estão apresentadas na Tabela 1.

    Ferramentas Utilizadas

    Tabela 1 - Ferramentas Utilizadas no Projeto

    Ferramenta Finalidade
    GitHub Organizar, versionar e documentar artefatos produzidos para o projeto.
    Teams Realizações de reuniões e gravações de apresentações.
    Figma Produção de artefatos gráficos.
    MkDocs Criação das páginas de documentação.
    Visual Studio Code Edição dos arquivos de documentação.
    WhatsApp Utilizado como principal canal de comunicação.
    YouTube Hospedagem de vídeos produzidos.
    Excel Criação de planilhas relacionadas ao cronograma e horários.
    Draw.io Produção da primeira versão do Rich Picture.
    Google Drive Compartilhamento de arquivos.
    This Person Doesn't Exist Criação de imagens falsas para as personas.

    Fonte: Autores.
